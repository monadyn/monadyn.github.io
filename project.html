Research and Project

  • Fasion AI: Knowledge aided meta inference, NLP/NLG (product description generation), Image Aesthetics Assessment, CTR prediction, content-based recommender system, quality evalution of multi-media content(text, image, video) etc.

  • Intelligent Infrastructure & Operation: the operation efficiency in container clouds platform, such as anomaly detection, root-cause diagnosis (e.g., eDiagnosis), trend prediction, adaptive alerting, autoscaling, resource management and scheduling etc, by the approaches of statistical learning (Machine learning and Deep learning) and time series operational data analysis.

  • Measurement & Performance Analysis: In a large-scale distributed system, log files of all the involved components are distributed in different locations. I developed an automatic log analysis tool (e.g., iPerf ), which use fine-grained correlation analysis of performance metrics from infrastructure layer (e.g., TShark, Wireshark, OProfile, Collectl) to application layer (e.g., Apache, Tomcat, MySQL) in timeline graphs, can pinpoint the root-cause of performance issues efficiently.

  • Distributed System Security: Large scale systems typically adopt distributed architecture, such as n-tier systems for Web applications. Due to complex dependency among the distributed nodes, there exists potential performance issue enigmas with hard traceability. I dig into those performance vulnerabilities (e.g., VSI-DDoS Attacks and Tail Attacks) to motivate Web services of fast responsiveness.

  • Security in Cloud Computing: Cloud Elasticity (e.g., Amazon EC2) provides opportunity of auto-scaling computing resources for dynamical user requirements. However, performance interference due to resource contentions among co-located VMs block the promotion of Cloud computing. We investigate those performance vulnerabilities inside the cloud (e.g., Tail Amplification in my thesis, ) to protect cloud services of fast responsiveness.

  • Virtual Cluster Management: With the wide adoption of Cloud Computing for today's enterprise and research institution, the end-users expect the out-of-box applications. SIMULOCEAN is a web-based computing platform for scientiﬁc applications on Container technology, such as Docker, Singunarity. Those contained applications can run in multi-cloud environments (PEARC'17, Gateways'16), such as Private Cloud, Public Cloud Amazon ECS, HPC Cloud XSEDE.

  • Infrastructure Software: Good infrastructure layer in large software sytems should provide easy traceability for debugging and tracing performance. My individual patent is a representative tracing tool in mobile phones, that can capture the critical debug informations on the fly anywhere and anytime for testing engineers.

  • Build and Test Automation: Continuous integration and continuous delivery are efficient to bug-free software developments. iBuild and iTest is an automatic and parallel build and test platform, serving in real production environments in Spreadtrum Inc, reducing the building time from hours to minutes level, and finishing the regression test of tens of thousands test cases within 20 minutes.

  • Data Mining Systems: Mining big data is useful for decision-making in companies. I analyze the work efficiency of each engineer from Bugzilla database for project managers (iBugzilla). We automatically detect abnormal logs to capture the potential problems through matching the critical events for developers (iLog).
